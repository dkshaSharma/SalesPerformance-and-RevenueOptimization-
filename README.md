# SalesPerformance-and-RevenueOptimization-
An Excel Analytics Personal Project

Project Overview:
This project is a Sales Performance & Revenue Optimization Dashboard built using Microsoft Excel, based on a full year (January–December 2024) of simulated e-commerce data with over 500 orders. It includes multiple sheets such as raw data, monthly trends, category, regional, and product-level analysis, all connected through 90+ formulas without any hardcoding. The core idea behind the project is to analyze a decline in monthly revenue for an e-commerce business by identifying patterns, trends, and key performance drivers, and presenting them in a clear, visual dashboard that is easy for anyone to understand.

What problem does it solve:
The dashboard helps answer critical business questions when revenue is falling. It identifies which months perform best or worst, which product categories generate the most profit, and which regions are underperforming. It also highlights top-performing and low-performing products, along with the overall profit margin of the business. By providing these insights in a structured and visual way, it enables decision-makers to quickly understand the reasons behind revenue decline and take informed actions.
